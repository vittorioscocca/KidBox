One-line: The folder tile used in Documents — neutral surface, folder icon, title, "Apri categoria" hint and an optional sync pill.

```jsx
<CategoryCard title="Referti medici" syncLabel="Sincronizzato" syncTone="green" onClick={open} />
```

`syncTone`: green | orange | red. Omit `syncLabel` to hide the pill. Grid these 2-up like HomeCard.
