# KidBox — Website UI Kit

Recreation of the marketing landing page at `KidboxLanding/public/index.html`
(kidbox.app) — nav, hero with App Store CTA, stats bar, and the five-AI-agent
grid. Colors and DM Sans typography are pulled straight from the source
`:root` palette and now live in the shared design-system tokens.

Only the hero + agents sections are recreated here (above-the-fold + the
page's signature section); the source file also has Proactive/Features/Plans/
Footer sections in the same visual language if you want to extend this kit.
